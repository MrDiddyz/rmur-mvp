# MurMur Agents

## MurMur Spectre Agent
Central orchestrator of the MurMur constellation.

The Spectre Agent observes all agents,
routes tasks, coordinates collaboration,
and ensures the creative pipeline flows correctly.

## Composer Agent
Generates musical ideas and motifs.

## Arrangement Agent
Structures and develops the composition.

## Voice Agent
Handles vocal generation and lyrical delivery.

## Training Agent
Improves models and evaluates results.
