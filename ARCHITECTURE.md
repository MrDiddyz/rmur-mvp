# MurMur Architecture

MurMur operates as a constellation of specialized agents.

At the center is the **MurMur Spectre Agent**.

The Spectre Agent:

- receives prompts
- distributes tasks
- coordinates agent collaboration
- evaluates outputs

Pipeline:

Prompt
 ↓
Spectre Agent
 ↓
Composer Agent
 ↓
Arrangement Agent
 ↓
Voice Agent
 ↓
Output
